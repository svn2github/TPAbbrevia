TAbZipper.DeleteTaggedItems

TAbZipper

procedure DeleteTaggedItems;

Finds all files that have their Tagged property set to True and marks them to be deleted.

The next time the archive is saved, all marked files and their corresponding information are deleted from the archive.

See the TagItems method of the TAbBaseBrowser class in the printed documenation for information about how tagging works.

See also: DeleteFiles, Save, TAbBaseBrowser.ClearTags, TAbBaseBrowser.OnChange, TAbBaseBrowser.TagItems, TAbBaseBrowser.UnTagItems
