IZipKit.CompressionMethodToUse

IZipKit

object.CompressionMethodToUse
The object is always an IZipKit.

Selects the compression method used when adding or freshening items in the archive.

The possible values for CompressionMethodToUse are:

Option
Description
smStored
The file is stored without any compression.
smDeflated
The file is compressed using the deflate method.
smBestMethod
The file is deflated, but if the resulting file is larger than the original, the file is simply stored.
