TAbDirDlg Class

Hierarchy—Properties—Methods

Unit:

AbDlgDir

Description:

The TAbDirDlg class provides a dialog for selecting a directory.

Windows

The TAbDirDlg class allows you to use the Windows shell browser dialog to browse the shell for a folder.

Linux

The TAbDirDlg class creates a custom form that display a "TreeView" style hierarchy of folders to browse the file system for a folder.
