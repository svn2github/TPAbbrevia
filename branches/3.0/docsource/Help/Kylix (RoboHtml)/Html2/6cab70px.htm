TAbMakeCab.LZXWindowSize

TAbMakeCab

property LZXWindowSize : Integer;

Default: 18

Sets the window size for the LZX compression engine.

The LZX window size must be a power of 2, from 215 to 221. The greater the window size, the greater the compression. This property sets the exponent to which the base (2) will be raised and must be between 15 and 21. The following table illustrates the effect of this property setting on the LZX window size:
